# RTW88-auto-report

An auto report bash script to test all RTW88 supported devices.

Test report includes:

1) Kernel Version
2) CPU Info
3) System Info
4) USB Tree
5) USB Device Info
6) SDIO Tree
7) SDIO Info
8) WIFI supported channels and internal Info
9) Network Speed Test
10) DNS Ping Test
11) Self Ping Test
12) IPERF3 DUT <-> Router <-> HOST Test
13) IPERF3 DUT <-> HOST Test
